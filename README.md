# 日语刷词 PWA 使用说明

## 简介

日语刷词 PWA 是一款基于网页技术的轻量级刷词（闪卡记忆）应用，兼容移动端访问。旨在帮助用户通过导入 CSV 词库，实现日语单词的快速记忆与复习。

---

## 功能概览

### 1. 搜索模式
- **CSV 导入**：支持导入符合格式的 CSV 文件，将词库存储至浏览器本地缓存。
- **智能搜索**：支持按日语（平假名/片假名）、中文释义、读音罗马音进行模糊搜索。
- **语音播放**：每个搜索结果右侧配备语音播放按钮，可即时聆听单词发音。

### 2. 刷词模式
- **随机打乱**：点击"打乱并开始"可对当前词库进行随机排序，打破记忆顺序。
- **3D 翻页卡片**：卡片正面展示日语单词，点击卡片或"下一个"切换单词并触发 3D 翻转动画显示中文释义。
- **语音播放**：卡片正面右上角悬浮语音按钮，点击即可聆听当前单词发音。
- **进度追踪**：顶部实时显示当前刷词进度（如 `3 / 50`）。
- **无缝切换**：支持"上一个"、"下一个"按钮快速切词。

### 3. 数据持久化
- **一次导入，永久生效**：词库导入后会自动保存至浏览器 `localStorage`，刷新或重新打开页面无需重复导入。

---

## CSV 格式要求

导入的 CSV 文件需符合以下格式（以逗号分隔）：

```csv
平假名,片假名,中文释义,读音（罗马音）
```

**示例：**

```csv
きょうと,キョート,京都,kyou to
さくら,サクラ,樱花,sakura
```

> 注：读音（罗马音）为选填字段。导入时会对空行和无效行进行过滤。

## 使用流程

### 首次使用

1. 准备好符合格式的 CSV 词库文件。
2. 打开应用，进入**搜索模式**，点击**导入 CSV** 按钮选择文件。
3. 导入成功后，词库会自动保存至本地。
4. 点击顶部的**刷词模式**切换至刷词界面。
5. 点击**打乱并开始**进入刷词流程。

### 日常使用

1. 再次打开应用时，已导入的词库会自动加载。
2. 可直接在搜索框查词，或切换到刷词模式复习。

## 界面特色

- **主题配色**：采用现代蓝白配色方案（`#4a90e2` 主色调）。
- **3D 翻转动画**：使用 CSS `perspective` 与 `transform: rotateY()` 实现真实的卡片翻转效果。
- **触摸友好**：按钮与卡片设计均考虑了移动端触摸交互体验。
- **空状态提示**：无词库或搜索无结果时，提供友好的引导文案。

## 注意事项

1. **网络环境**：语音播放功能依赖 Google Translate TTS，需保持网络畅通。
2. **浏览器兼容**：建议使用最新版 Chrome、Safari 或 Edge 浏览器以获得最佳体验。
3. **数据安全**：词库仅存储在本地浏览器中，清除浏览器数据将导致词库丢失。
4. **CSV 编码**：请确保 CSV 文件编码为 UTF-8，以避免中文乱码。
