# 日语刷词 PWA 使用说明

## 简介

日语刷词 PWA 是一款基于网页技术的轻量级刷词（闪卡记忆）应用，兼容移动端访问。旨在帮助用户通过导入 CSV 词库，实现日语单词的快速记忆与复习。

---

## 功能概览

### 1. 词库管理 (New!)
- **多词库支持**：支持创建多个独立的词库（如“N1词汇”、“生活用语”等）。
- **词库重命名 (New!)**：支持修改现有词库名称。特别地，若将“默认词库”重命名，系统会自动创建一个新的空白“默认词库”。
- **词库创建与删除**：可随时新增或删除词库。
- **独立导入与导出 (New!)**：支持为每个词库单独导入 CSV 文件，或将现有词库导出为 CSV 备份。
- **词库切换**：一键切换当前使用的词库，搜索和刷词将自动同步。

### 2. 搜索模式
- **智能搜索**：支持按日语（平假名/片假名）、中文释义、读音罗马音进行模糊搜索。
- **语音播放**：每个搜索结果右侧配备语音播放按钮，可即时聆听单词发音。
- **状态提示**：顶部显示当前正在使用的词库名称。

### 3. 刷词模式
- **选择性复习**：基于当前选定的词库进行刷词。
- **错题本功能 (New!)**：刷词时点击“记不住 ❌”或按下 `M` 键，单词将自动加入专门的“错题本”库，方便后续精准复习。
- **多种刷词模式**：
  - **日 → 中**：正面日语单词，背面中文释义。
  - **中 → 日**：正面中文释义，背面日语单词。
  - 随时点击“模式切换”按钮进行反转。
- **键盘快捷键**：
  - `空格`：翻转卡片（显示/隐藏答案）。
  - `右箭头 →`：下一个单词。
  - `左箭头 ←`：上一个单词。
  - `V 键`：播放当前单词语音。
  - `M 键`：标记为记不住（加入错题本）。
- **语音自动播放**：切换到下一个卡片时自动播放日语读音，可在顶部开启/关闭。
- **随机打乱**：点击"打乱并开始"可对当前词库进行随机排序。
- **3D 翻页卡片**：卡片正面展示日语单词，点击卡片或"下一个"切换单词并触发 3D 翻转动画显示中文释义。

### 4. 系统设置 (New!)
- **深色模式**：点击右上角月亮/太阳图标切换主题，保护视力。
- **本地存储**：所有词库数据及个人偏好（模式、自动播放、主题）均保存在浏览器 `localStorage` 中，持久化存储。
- **向下兼容**：自动将旧版本的单一词库迁移为“默认词库”。

## CSV 格式要求

导入的 CSV 文件需符合以下格式（以逗号分隔）：

```csv
平假名,片假名,中文释义,读音（罗马音）
```

**示例：**

```csv
きょうと,キョート,京都,kyou to
さくら,サクラ,樱花,sakura
```

> 注：读音（罗马音）为选填字段。导入时会对空行和无效行进行过滤。

## 使用流程

### 首次使用

1. 准备好符合格式的 CSV 词库文件。
2. 打开应用，进入**搜索模式**，点击**导入 CSV** 按钮选择文件。
3. 导入成功后，词库会自动保存至本地。
4. 点击顶部的**刷词模式**切换至刷词界面。
5. 点击**打乱并开始**进入刷词流程。

### 日常使用

1. 再次打开应用时，已导入的词库会自动加载。
2. 可直接在搜索框查词，或切换到刷词模式复习。

## 界面特色

- **主题配色**：采用现代蓝白配色方案（`#4a90e2` 主色调）。
- **3D 翻转动画**：使用 CSS `perspective` 与 `transform: rotateY()` 实现真实的卡片翻转效果。
- **触摸友好**：按钮与卡片设计均考虑了移动端触摸交互体验。
- **空状态提示**：无词库或搜索无结果时，提供友好的引导文案。

## 注意事项

1. **网络环境**：语音播放功能依赖 Google Translate TTS，需保持网络畅通。
2. **浏览器兼容**：建议使用最新版 Chrome、Safari 或 Edge 浏览器以获得最佳体验。
3. **数据安全**：词库仅存储在本地浏览器中，清除浏览器数据将导致词库丢失。
4. **CSV 编码**：请确保 CSV 文件编码为 UTF-8，以避免中文乱码。
